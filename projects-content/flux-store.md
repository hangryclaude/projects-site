# Flux Store

A polished e-commerce storefront for curated minimal tech accessories — ambient lighting, desk gadgets. Full shopping flow: product grid, cart, checkout, order-success page, all wrapped in framer-motion transitions. There is no inventory and no payment processor; the store is the product.

## How it works

Next.js App Router with a page per flow step (`/` → `/cart` → `/checkout` → `/success`). Cart state persists via Zustand.

## Stack

TypeScript, Next.js 16, React 19, Zustand, framer-motion, Tailwind CSS 4

## Status

Frontend prototype. Looks like a store, sells nothing.
