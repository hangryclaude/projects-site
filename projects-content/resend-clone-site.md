# Resend Clone

A high-fidelity frontend reproduction of Resend — the landing page and the internal dashboard, including email sending and management workflows. Built while living inside the real product daily for the email machine; cloning it was the fastest way to understand what makes its UI good.

## How it works

Static HTML/CSS/JS, no framework. The dark-mode aesthetic is reproduced with CSS variables; dashboard views cover sending, blasting and account setup, plus a couple of original utility pages (a provider finder and a scale planner) bolted into the same shell. Open `index.html`, done.

## Stack

HTML, CSS, vanilla JavaScript

## Status

Working static clone. A study, not a product.
