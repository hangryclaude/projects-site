# Website Builder

A hustle with Ethan: AI agents find local businesses with no website on Google Maps, auto-generate them a site, and pitch it to them for $50. The lead source and the product are the same query — if they're not online, they're both findable and sellable.

## How it works

An agent pipeline: discover no-website businesses on Maps, generate a site per business, then pitch. Collaboration rules and project context live in the repo's `CLAUDE.md`.

## Stack

TypeScript, Claude agents.
