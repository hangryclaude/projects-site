# Animations

Blender asset dump for the madebyangus portfolio: 3D models, rendered stills, and the helper scripts that produced them.

## How it works

Not an app — a working directory under version control so the renders feeding other projects are reproducible. Models in `models/`, output in `renders/`, Blender Python scripts in `scripts/`.

## Stack

Blender, Python

## Status

Asset repo. Used by madebyangus; grows when that does.
