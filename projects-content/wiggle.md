# Wiggle

An AI tool that drafts the "can I please…" text to your adults — parent, teacher, boss — written to actually land. Honest persuasion is the whole point: it makes a real case, it doesn't lie or manipulate.

## How it works

You describe the situation; a Next.js app sends it through Claude and returns a respectful, well-argued draft you can send as-is.

## Stack

TypeScript, Next.js 16, React 19, Tailwind CSS 4, Anthropic SDK

## Status

Working prototype. Needs your own API key.
