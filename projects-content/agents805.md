# 805 Agents

A landing page for a local pitch: AI agents built and installed in person for Santa Barbara, Goleta, and Carpinteria businesses. Flat $500 setup, then a monthly rate per agent. The 805 area code as a wedge — show up in person, install the thing, no SaaS runaround.

## How it works

One hand-built `index.html`. Editorial design — Fraunces serif over Hanken Grotesk, warm paper palette, an SVG film-grain overlay — with no framework and no build step.

## Stack

Single-file HTML/CSS, Google Fonts

## Status

Landing page — the local-agent offer, waiting on outreach.
