# nox-explode

Click anywhere, detonate a particle explosion in slow motion. There's a slider for how slow (0.03x to 1x), plus blast force and particle density dials, and a Replay button that re-runs the last detonation with the new settings.

## How it works
One self-contained HTML file — vanilla JS on a fullscreen Canvas 2D, control panel in CSS backdrop-filter. No framework, no build, no network calls. `open index.html` and start clicking.

## Stack
HTML, vanilla JS, Canvas 2D

## Status
