# Made by Angus v9

The continuation of v8's 3D-laptop portfolio — same click-the-laptop, fly-into-the-screen, browse-30-real-client-sites concept. The README still says v8; the version number moved faster than the docs, which tells you everything about the pace these shipped at.

## How it works

Same architecture as v8: vanilla Three.js with a CSS3DRenderer iframe locked to the laptop screen, a depth-only plane punching the WebGL canvas transparent, GSAP camera tweens, and pointer-event gating between canvas and iframe.

## Stack

TypeScript, React, Three.js, GSAP, Vite
