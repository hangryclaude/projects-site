# Site Forge

Five complete, opinionated web design systems built by hand as raw material for a website generator. The bet: a generator that picks from finished design systems beats one that improvises CSS.

## How it works

Each theme is a full standalone page with fonts, palette, motion, and a signature move baked in: Editorial Luxe (serif, cream paper), Neon Dark SaaS (glowing orbs, glassmorphism), Brutalist Studio (hard borders, mono type), Soft & Playful (gradient mesh, blobs), and Aether ("break physics"). A gallery index previews all of them in scaled-down live iframes. Next step was wiring them into a generator that picks a system per prompt and fills it with real content — that step became other projects.

## Stack

HTML, CSS, vanilla JS

## Status

Reference library, local. Superseded by the crazy-web-effects skill factory.
