# heynox-referral

A one-file NOX fan referral page: full landing site — hero, platform strip, six feature cards, Free/Pro pricing, download CTA — with a single `REF_LINK` constant to swap in your tracked referral code. Footer says what it is: "Not affiliated fan referral page."

## How it works
One 12KB `index.html`, zero dependencies. All CSS inline (glow blobs, gradient text, card hovers), a ten-line script that wires the main CTA to the referral link and fires an optional Plausible ping. Deploys anywhere that serves static files.

## Stack
HTML, CSS, vanilla JS

## Status
Working template; local only, never deployed.
