# heynox-proto

Landing-page prototypes for the NOX pitch. The main one opens with "Everyone is **screaming** at you." and resolves to "One calm **inbox**." — scroll-reveal typography selling the unified-inbox story.

## How it works
Self-contained HTML files, no build step: a hero page, a 2D runner variant, and a `flagship/` folder with three concept builds (arc, hero figure, POV glass). Each is one file you open in a browser.

## Stack
HTML, CSS, vanilla JS

## Status
Prototype — design explorations, never the shipped site.
