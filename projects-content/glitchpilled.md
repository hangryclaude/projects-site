# Glitchpilled

Brand art generator for a "glitchpilled" persona — a banner and profile picture in a Limitless/NZT aesthetic: deep blue-black, gold-amber glow, cyan glitch. No README; the code is the spec.

## How it works

A single Python script using PIL builds the images procedurally — a vertical base gradient, soft radial glow blobs, and layered type in Arial Black over the top, with a seeded RNG so the output is repeatable. Outputs `banner.png` and `pfp.png`.

## Stack

Python, Pillow (PIL)

## Status

A one-off asset script. Ran once, made the art, done.
