# Underground 805 Theme

The front door for Underground 805 — an enter-page and landing site for the brand, heavy on atmosphere: noise overlay, scanlines, vignette, a blinking "● LIVE", `EST. 805 / SANTA BARBARA / CA` in mono caps.

## How it works

Static HTML/CSS/JS, no framework. An entry gate section leads into the main page; Anton for the shouty headlines, Space Mono for the terminal bits, Inter for body.

## Stack

HTML, CSS, JavaScript, Google Fonts.

## Status

Working standalone; the real storefront lives in the Shopify theme inside UndergroundBrandd.
