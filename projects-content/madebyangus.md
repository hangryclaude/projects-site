# Made by Angus

The portfolio site for madebyangus.com — plus four complete fake agencies living in its Labs section. Everything hand-built: no framework, no build step, no image assets.

## How it works

One main page and four showcase microsites (an editorial agency, a SaaS uptime tool, a coffee roaster shop, a Swiss-minimalist architecture studio), each deliberately a totally different aesthetic — that's the point of the Labs section. Project cards use pure CSS-art media elements instead of screenshots, so the whole site ships with zero images. One JS file drives the cursor, clock, scroll-reveal, and a live counter. Type comes from ten Google Fonts families in different combinations per page.

The current v7 adds the interactive layer: a crane word-box, a code-slit tug-of-war, and an interactive ring — small physical toys built into the page rather than decoration.

## Stack

Plain HTML, CSS, vanilla JS, Google Fonts
