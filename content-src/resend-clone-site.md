# resend-clone-site
High-fidelity landing page and dashboard clone of Resend.

A comprehensive frontend reproduction of the Resend platform, featuring a polished landing page and a detailed internal dashboard interface including email sending and management workflows.

## Features
- **Complete Dashboard Suite**: Dedicated views for sending, blasting, and account setup.
- **High-Fidelity UI**: Precise implementation of Resend's dark-mode aesthetic using CSS variables.
- **Interactive Tools**: Integrated utility pages including a provider finder and scale planner.
- **Responsive Layouts**: Modern CSS architecture tailored for a seamless professional application feel.

## Run
Since this is a static site, open `index.html` directly in any web browser.

## Stack
HTML, CSS, and JavaScript.
