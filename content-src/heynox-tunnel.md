# heynox-tunnel
Interactive Three.js/WebGL tunnel ad for RPLY — fly the notification tunnel to the Nox vault

An immersive WebGL experience designed as a promotional advertisement for RPLY. It features a high-performance 3D tunnel environment that users navigate through to reach the Nox vault.

## Features
- High-performance 3D rendering using Three.js
- Interactive "fly-through" navigation mechanics
- Responsive WebGL viewport for cross-device compatibility
- Optimized asset pipeline via Vite

## Run
```bash
npm run dev
```

## Stack
Three.js, Vite, WebGL.
