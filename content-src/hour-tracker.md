# hour-tracker

**Live:** https://hour-tracker-smoky.vercel.app

A browser-based time tracker styled as a space mission launch console.

## What it does

Opens a single HTML page themed around a rocket launch ("Mission 162"). The UI runs entirely in the browser with animated particle effects, shooting stars, and a nebula background — all rendered via CSS and canvas. No server, no build step, no dependencies to install.

## Use

```bash
open index.html
# or serve locally:
python3 -m http.server 8080
```

Then visit `http://localhost:8080` in any modern browser.

## Stack

Vanilla HTML/CSS/JS — single file, zero dependencies, no paid API.
