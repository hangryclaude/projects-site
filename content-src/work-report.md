# work-report
Automated weekly work report generator from git commit history

This tool parses git commit logs to generate a professional, styled HTML weekly summary of development activity.

## Features
- Converts raw git commit history into a structured report.
- Generates a modern, dark-themed HTML layout.
- Organizes work by date ranges.
- Provides a clean, responsive presentation for stakeholders.

