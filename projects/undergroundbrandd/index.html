UndergroundBrandd

GitHub ↗

The complete operating manual for Underground 805, my Santa Barbara streetwear brand — not a mood board, the actual runbook. Roughly 60 playbook docs covering everything between "idea" and "drop day."

How it works

One repo, four layers. A full Shopify theme (Dawn-based, Liquid) with custom code and SEO schema snippets. Marketing playbooks: Meta and TikTok ads step-by-step, Klaviyo flows, influencer hit lists with outreach DMs, a drop-day runbook scripted hour by hour. Ops: Alibaba supplier trackers, profit-margin and pricing CSVs, size charts, shipping breakdowns, legal setup. Plus a working SMS sender (Node) with subscriber list and templates.

Stack

Liquid, Shopify theme, Node SMS scripts, Markdown + CSV playbooks.

Status

Working as a knowledge base; the brand itself is pre-launch.
