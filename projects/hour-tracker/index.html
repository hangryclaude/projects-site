A browser time tracker styled as a space mission launch console — tracking study hours as "Mission 162", because a plain timer wasn't going to get used.

How it works

One HTML page, everything client-side: animated particle effects, shooting stars, and a nebula background rendered with CSS and canvas. No server, no build step, no dependencies.

Stack

Vanilla HTML/CSS/JS, single file, Vercel

Status

Live at hour-tracker-smoky.vercel.app.
