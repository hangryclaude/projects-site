Claude Anim

Web & creativePython2026
GitHub ↗

The Claude logo as a five-phase terminal animation. Petals ignite outward, the mark drifts on a lissajous path with comet trails, satellites orbit and merge, the CLAUDE wordmark wipes in, then it breathes and fades.

How it works

Pixel-art rendering with ▀ half-block characters (two vertical pixels per terminal cell) and truecolor ANSI in the brand's warm orange palette. Ctrl-C restores the cursor cleanly.

Stack

Pure Python 3 stdlib — no installs, no API

Status

Working. Run python3 claude.py and watch.
