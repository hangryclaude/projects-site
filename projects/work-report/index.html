Turns raw git history into a styled weekly work summary — the "what did I actually ship this week" report, generated instead of written.

How it works

Parses git commit logs and organizes them by date range into a modern dark-themed, responsive HTML page (like this-week.html) from a commits.txt dump. Meant to be readable by someone who isn't going to read a git log.
