Kairos GP

School & learningHTML2026
GitHub ↗

The Kairos classroom tracker reimagined as a Grand Prix. Each student is a race car; good work moves them around the track, onto the podium, and up a live leaderboard the whole class watches on the board.

How it works

Single-file web app with Firebase Firestore for real-time sync — standings update across devices as points land. Students get deep car customization (livery, style), there are two point types that convert into track position, streak tracking rewards consistency over one-off wins, and per-student stat charts show progress over time. A class sound board plus keyboard shortcuts let the teacher run it fast from the front of the room, and confetti celebrations make kids want the board on. A firebase-setup.html walks anyone through wiring their own project.

Stack

Single-file HTML/CSS/JS, Firebase Firestore, Netlify

Status

Working — the rocket-themed kairos-tracker became the main line.
