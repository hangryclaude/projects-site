Prompting Course

School & learningHTML2026
GitHub ↗

A self-paced course on prompting and Claude, built as one HTML page. Five modules: what an LLM actually is, prompting fundamentals, prompting like Anthropic's own engineers, prompting Claude Code (agents, not chat), and free official courses with certificates.

How it works

Single dark-themed page with embedded videos, exercises, and curated links per lesson. Checkboxes track completion in the browser with a sticky progress bar; finish everything and a done banner appears. No framework, no server — the whole course is index.html.

Stack

Single-file HTML/CSS/JS, embedded YouTube

Status

Working — a personal curriculum, kept around as a reference.
